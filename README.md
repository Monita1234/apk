# App Pre-Serie DJ (Flutter)

MVP listo para correr **sin backend** (usa SharedPreferences). Incluye:
- Login (invitado y admin simulado).
- Catálogo de equipos.
- Detalle con comentarios.
- Reservas con validación de choque de fechas.
- Mis reservas.
- Panel admin para crear/editar/borrar equipos.

## Cómo correr
```bash
flutter pub get
flutter run
```
> Probado en Flutter 3.x.
